What You'll Learn

Information Technology Assistant (150 hours)

In this course, you’ll master digital literacy and learn to create web pages using HTML and CSS. You'll build projects such as blogs, schedules, and portfolio websites while developing critical soft skills like communication and time management.

You'll also gain hands-on experience with Git and GitHub for version control and collaboration.

Key Projects: Fashion Blog, Daily Buzz Website, and a personalized final project.

Certifications: Florida Soft Skills Credential.

Web Design Foundations (150 hours)

In this course, you’ll use HTML, CSS, and Adobe Photoshop to create user-friendly websites while preparing for the Adobe Certified Professional exam. Learn responsive web design, optimize graphics, and tackle real-world coding challenges to build your portfolio.

Projects include a QR code component, blog preview card, social links profile, and a recipe page, all designed to strengthen your front-end development skills. By the end, you’ll have practical experience, a professional portfolio, and an industry-recognized certification.

Web Interface Design (150 hours)

Enhance your design skills with advanced CSS, UI/UX principles, and responsive web design. This course covers Flexbox, Grid, CSS animations, and prototyping in Figma to create user-friendly, visually stunning websites.

Projects include a Flexbox To-Do App, the Tea Cozy business website, a Figma prototype for user testing, and real-world challenges like the Product Preview Card. By the end, you’ll have hands-on experience, a strong portfolio, and the Knowledge Pillars HTML/CSS Coding Specialist certification.

Web Scripting (150 hours)

Learn JavaScript to build dynamic, interactive websites. This course covers core programming concepts, DOM manipulation, APIs, and debugging techniques to create responsive web applications.

Projects include the Chore Door game, a time-tracking dashboard, a tip calculator app, and a newsletter sign-up form with success message functionality. By the end, you’ll be ready to develop real-world applications and earn the Knowledge Pillars JavaScript Coding Specialist certification.

Web Media Integration (150 hours)

Master back-end web development with PHP, SQL, and MySQL to build dynamic, data-driven websites. This course covers writing PHP for logic and form handling, designing relational databases, and integrating dynamic content for seamless user experiences.

Projects include the FizzBuzz Challenge, the Repetitive Cafe website, the Bobby Tables database project, and Save the Farm! interactive functionality. By the end, you’ll be ready to develop full-stack applications and earn the Knowledge Pillars PHP Coding Specialist and SQL Coding Apprentice certifications.

Web E-commerce (150 hours)

Learn to build and manage e-commerce websites that drive sales and enhance user experience. This course covers digital marketing, SEO best practices, and web accessibility while integrating Google Analytics for data-driven insights.

By the end, you’ll be able to design professional, accessible e-commerce platforms and earn the Google Analytics 4 Certification.

Web Interactivity (150 hours)

Bring websites to life with interactive elements and modern frameworks. This course covers building and managing sites with WordPress, implementing advanced tools for functionality, and ensuring quality through front-end testing.

By the end, you’ll be able to create dynamic, user-centered websites and earn the CIW Advanced HTML5 & CSS3 Specialist and Knowledge Pillars WordPress Certified Editor certifications.
